# 催促くん - お金の貸し借りを自由に - 

## アプリ概要
お金の貸し借りを管理し、お金を貸した後のストレス（返金を催促）を少なくするiOSアプリ

## 使用技術・ライブラリ
- SwiftPackageManeger（ライブラリ管理）
- Firebase(store,Auth,storage)
- Alamofire
- QRScanner
- PKHUD

## アーキテクチャ・開発方法
- MVVM (Model View ViewModel)
- チケット駆動開発
